Anatomía de un fichero CSS

Dentro de la etiqueta <head> utilizamos la etiqueta <link> para indicar la ruta del fichero con nuestros estilos css.
Por ejemplo: <link rel="stylesheet" href="css/styles.css">

body {
     background-color: #eaf6f6;
}


Colores

Utilizamos la propiedad color. Por ejemplo:

h2 {
     color: #66bfbf;
}
