Anatomía de un fichero CSS

Dentro de la etiqueta <head> utilizamos la etiqueta <link> para indicar la ruta del fichero con nuestros estilos css.
Por ejemplo: <link rel="stylesheet" href="css/styles.css" type="text/css">

body {
     background-color: #eaf6f6;
     background-image: url(../images/bkg-css.jpg);
}


Colores

Utilizamos la propiedad color. Por ejemplo:

h2 {
     color: #66bfbf;
}

h6 {
     color: rgba(178, 76, 127, 1);
}


Selectores

Utilizamos para habilitar selectores las propiedades class y también id.

     La propiedad class nos permite agrupar elementos de uno o varios selectores bajo una misma clase. Por ejemplo:

     .paragraph {
         color: #1f7c76;
     }

     La propiedad id nos permite diferenciar individualmente elementos de un mismo selector o clase. Por ejemplo:

     #firstParagraph {
         color: #163331;
     }


El selector de tipo id tiene más importancia que el selector de tipo class. El selector de tipo class tiene más importancia que el selector general.
Un elemento puede tener varias clases asociadas, pero sólo una id.

Es posible utilizar un selector con eventos. Por ejemplo:

     .changeParagraph:hover {
         color: darkblue;
     }


Podemos utilizar un selector de etiqueta dentro de un selector de id o de clase. Por ejemplo:

     thirdParagraph em {
         color: red;
     }


Podemos utilizar un selector de atributo. Por ejemplo:

     input[type="text"] {
         background-color: #faf3dd;
     }

     input[type="submit"] {
         background-color: greenyellow;
     }




También tenemos el selector universal *. Por ejemplo:

     * {
         font-family: 'Times New Roman', Times, serif;
     }

Si agregamos la instrucción !important a una propiedad, indica que predomina sobre las demás que la sobreescriban. Por ejemplo:

     * {
         font-family: 'Times New Roman', Times, serif !important;
     }


Capas

Para crear capas, usamos la etiqueta <div> </div>. Todas las etiquetas contenidas bajo una capa, quedan agrupadas.
Por ejemplo:

     .paragraph {
         color: #1f7c76;
     }


Fuentes

Las propiedades más importantes del uso de fuentes son:

     font-family para indicar la familia de fuentes
     font-size para indicar el tamaño de la fuente
     font-weight para indicar el peso de la fuente (negrita)
     font-style para indicar el estil de la fuente (cursiva)

Por ejemplo:

     h5 {
         font-family: Impact, Haettenschweiler, 'Arial Narrow Bold', sans-serif;
         font-size: 12px;
         font-weight: bold;
         font-style: italic;
     }

Podemos utilizar fuentes externas mediante el uso de la etiqueta <link>.
Por ejemplo: <link href="https://fonts.googleapis.com/css2?family=Ranchers&display=swap" rel="stylesheet">

     Testeando la fuente

     #font1 {
         font-family: 'Ranchers', cursive;
         }

Podemos importar fuentes mediante la instrucción @font-face. Por ejemplo:

     Testeando la fuente

     @font-face {
         font-family: 'Roboto Mono Custom'; -> damos nombre a la fuente
         src: url(../fonts/Roboto_Mono/RobotoMono-Regular.ttf); -> ruta local de la fuente
     }
