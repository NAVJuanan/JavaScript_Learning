Anatomía de un documento HTML

El documento se inicia con el elemento <!DOCTYPE html> que nos indica que es un documento HTML5. Es un elemengo de legado histórico que hay que incluir.

A continuación tenemos el elemento <html lang="en"> </html> que es la raíz del documento y contendrá al resto de elementos.
Contiene el atributo lang para indicar el idioma del contenido a los navegadores y motores de búsqueda. Dentro tenemos los elementos <head> y <body>.

El elemento <head> </head> es el contenedor de los parámetros no visibles como los metadatos, hojas de estilo, palabras clave para los motores de búsqueda, etc.
Contiene el elemento <title> </title> para indicar el título de la pestaña del navegador. Por ejemplo: <title>HTML5 Basics</title>
También contiene el elemento <meta> para indicar todo tipo de metadatos.
El atributo más importante es charset para indicar la codificación de la página. Por ejemplo: <meta charset="utf-8">
Otros elementos contienen los atributos name y content para especificar el tipo de información y el contenido. Por ejemplo: <meta name="author" content="Juan Antonio">

El elemento <body> </body> es el contenedor de todo lo que los visitantes de la página verán. El contenido de la página.


<!DOCTYPE html>
<html lang="en">
     <head>
          <meta charset="UTF-8" />
          <meta name="viewport" content="width=device-width, initial-scale=1.0" />
          <title>HTML5 Basics</title>
     </head>

     <body>
     </body>
</html>


Etiquetas para textos

<!-- Podemos introducir comentarios que no son visibles para los usuarios -->

Para introducir saltos de línea utilizamos la etiqueta <br>.
Para introducir espacios en blanco utilizamos la etiqueta &nbsp;.
Para introducir el carácter < utilizamos la etiqueta &lt;.
Para introducir el carácter > utilizamos la etiqueta &gt;.
Para introducir el carácter & utilizamos la etiqueta &amp.

Dentro de la etiqueta <p> </p> podemos escribir y agrupar párrafos.
Dentro de la etiqueta <center> </center> podemos centrar el texto.

Con la etiqueta <strong> </strong> escribimos en negrita.
Con la etiqueta <em> </em> escribimos en cursiva.

Para introducir una línea horizontal usamos la etiqueta <hr>, utilizando el atributo size para el tamaño y el atributo noshade para eliminar sombras. Por ejemplo: <hr size="3" noshade>

Con las etiquetas que van desde <h1> </h1> hasta <h6> </h6> podemos formatear encabezados.

Etiquetas para listas

Para crear una lista no ordenada utilizamos la etiqueta <ul> </ul> y con cada elemento la etiqueta <li> </li>

  • elemento 1
  • elemento 2
  • elemento 3

Para crear una lista ordenada utilizamos la etiqueta <ol> </ol> y con cada elemento la etiqueta <li> </li>

  1. elemento 1
  2. elemento 2
  3. elemento 3

Para crear una lista anidada utilizamos las etiquetas anteriores. Podemos utilizar el atributo type para seleccionar el estilo de lista. Por ejemplo: <ol type="a">

  1. elemento 1
  2. elemento 2
    • elemento 2.1
    • elemento 2.2
  3. elemento 3

Etiqueta para imágenes

La etiqueta para introducir imágenes es <img> con su principal atributo src indicando la ruta de la imagen.
Por ejemplo: <img src="img/230px-HTML5_logo_and_wordmark.svg.png" alt="html image" title="html5 image" height="100px" width="100px">



Etiqueta para enlaces

La etiqueta para introducir enlaces es <a> </a> con su principal atributo href indicando la ruta del enlace.
Por ejemplo: <a href="img/230px-HTML5_logo_and_wordmark.svg.png" title="open html image">Abrir imagen</a>
Por ejemplo: <a href="https://developer.mozilla.org/es/docs/Web/HTML" title="open mdn web" target="_blank">Abrir web</a>

Etiquetas para tablas

La etiqueta para introducir una tabla es <table> </table> donde podremos crear:
     Celdas de datos mediante la etiqueta <td> </td>
     Celdas-cabecera mediante la etiqueta <th> </th>
     Filas mediante la etiqueta <tr> </tr>
     Podemos encapsular la cabecera bajo la etiqueta <thead> </thead>
     Podemos encapsular el cuerpo bajo la etiqueta <tbody> </tbody>
     Podemos encapsular el pie bajo la etiqueta <tfoot> </tfoot>

Etiqueta para formularios

La etiqueta para crear formularios es <form> </form> donde podremos crear:
     Labels mediante la etiqueta <label> </label>
     Cajas de texto mediante la etiqueta <input type="text">
     Áreas de texto mediante la etiqueta <textarea cols="30" rows="3">
     Radio button mediante la etiqueta <input type="radio" value="yes">
     Select button mediante la etiqueta <select> </select> que incluirá las diferentes opciones
     Botones mediante las siguientes etiquetas:
         La etiqueta <button> </button>
         La etiqueta <input type="submit">
     Un selector de colores mediante la etiqueta <input type="color">
     Checkox mediante la etiqueta <input type="checkbox">
     Contraseñas enmascaradas mediante la etiqueta <input type="password">
     Calendarios mediante la etiqueta <input type="date">
     Rangos mediante la etiqueta <input type="range">
     Ficheros mediante la etiqueta <input type="file">
